## Getting Started

First you need to install [Node](https://nodejs.org/fr) v12.

### nhex-redux

You will also need to checkout the [nhex-redux project](https://github.com/tonai/nhex-redux).

Then run:
```
npm install
npm build
npm link .
```

### nhex-react

On nhex-react project run:
```
npm install
npm link nhex-redux
```

Then run project with:
```
npm run start
```
